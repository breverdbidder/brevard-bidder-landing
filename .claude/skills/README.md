# Claude Skills for BidDeed.AI

This directory contains Claude skills that enhance development workflows for the BidDeed.AI foreclosure auction intelligence platform.

## Installed Skills

1. **frontend-design** - Creates distinctive production-grade UIs
2. **skill-creator** - Builds custom Claude skills
3. **domain-name-brainstormer** - Generates available domain names
4. **content-research-writer** - Writes content with citations
5. **lead-research-assistant** - Finds qualified leads
6. **stripe-integration** - Implements Stripe payments

## Usage

Skills auto-activate when relevant. Optionally mention them:
```
"Use frontend design skill to improve the landing page"
"Use lead research assistant to find Brevard County investors"
```
