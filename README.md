# BrevardBidderAI Landing Page

Professional landing page for BrevardBidderAI - an agentic AI foreclosure intelligence platform.

## Tech Stack

- React 18
- Vite 5
- Tailwind CSS 3.4

## Quick Start

```bash
npm install
npm run dev
```

## Deploy to Vercel

### Option 1: Vercel CLI
```bash
npm i -g vercel
vercel
```

### Option 2: GitHub Integration
1. Push to GitHub
2. Connect repo at vercel.com/new
3. Auto-deploys on push

## Configuration

Update the demo video ID in `src/App.jsx`:
```javascript
const DEMO_VIDEO_ID = 'YOUR_YOUTUBE_VIDEO_ID';
```
